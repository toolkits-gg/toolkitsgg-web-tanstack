## Contributing

Contributions are welcome! Whether it's a bug report, a suggestion, or a pull request — all help is appreciated.

1. Read the [Contributing Guidelines](.github/CONTRIBUTING.md) and [Code of Conduct](.github/CODE_OF_CONDUCT.md).
2. Follow the [Local Setup](docs/LOCALSETUP.md) guide to get your environment ready.
3. Open a pull request or [create an issue](https://github.com/toolkits-gg/toolkitsgg-web/issues).

> [!TIP]
> First time contributing to open source? Don't be shy — come say hello in the [Discord](https://discord.gg/EgMaFJ8MYc).
